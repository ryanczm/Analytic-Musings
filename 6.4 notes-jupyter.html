Jupyter embedding as HTML in website:

Last updated: 7 Jan 2020 » Guide to embedding Jupyter as HTML in website »

Important CSS selectors to ctrl-f and modify:

  1. Pre-export:
    Stackoverflow: How do I remove gray border around Jupyter Notebook in HTML?
  2. Adjust/remove border: #notebook-container
  3. Adjust image sizing in mobile: div.output_area img, div.output_area svg AND div.output_subarea (controls image region)
  4. Adjust Python code font: .hl-ipython3
  5. Adjust H1-5: in-line styling
  6. MathJax: ctrl-f MathJax. Disable JS-CSS-HTML formatter extension to disable auto indent on save. It turns out that this extension's auto indent collapses the mathjax block into a single bar, thus undoing the effect. To remedy, I should get the correctly indented block and replace the collapsed block.
  7. H1/H2/H3: font-weight and font-size: vh. In-line styling only.
  8. Structure: body --> flexbox --> sidebar and main --> under main: navbar and content.
