Introduction

Note/Disclaimer - 20th August2021 : This is an expanded version of projects. For posterity, here is the original post. Interesting to see my perception of the world 9 months onwards (from when this post was originally written).

A list of topics or syllabus I want to learn in the long run. Why? TLDR: I realized how much I don't know. This hit me last December. I don't set myself a deadline, but what I do is try to set consistent cycles of work. Each topic gets a block of 7-14 days, and all the topic blocks form one cycle. For my own reference and progress tracking. Unfortunately, I like planning, so a common theme is me having completed only a tiny portion of stuff here, but then I end up planning out even more stuff/expanding this.
