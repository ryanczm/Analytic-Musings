Overview:

As per the title suggests, this is a collated, easy to use and navigate, spreadsheet of keyboard shortcuts/key commands I’ve collated over time as a reference, for:
Excel, Powerpoint, Word, Outlook, Premiere Pro, After Effects, Explorer, Chrome, Jupyter, VSCode, Git, and Tableau.
Here is the link to download the spreadsheet if you're lazy to read.

I started off learning the shortcuts for fun a while back from a standard list of shortcuts, and whenever I came across a useful one, I added to the list. It is quite comprehensive with 402(+/-) total shortcuts/commands. However, I can’t remember half of them and don’t really use half of them as well, and nobody really uses Explorer shortcuts. This might be a useful reference for Excel and Powerpoint especially if you need to use these tools heavily. If you know of a useful shortcut that isn’t inside, let me know!

Remarks

  1. Excel- Very useful. Not in any particular order, QAT will help. Quite a few useless shortcuts are in there, but I don’t want to remove them.
  2. Powerpoint- Very useful. QAT will help.
  3. Word- Not very useful beyond the standard shortcuts. QAT helps more.
  4. Outlook- Not useful, it’s just there because I decided to find Outlook shortcuts one day a long time ago, but it’s quite pointless.
  5. Premiere Pro/After Effects- Quite useful if you need to do editing or design work, but I forgot most of these because I don’t use them.
  6. Explorer- Not useful beyond several like F2 and Windows+E .
  7. Chrome- Quite useful, since everyone uses Chrome.
  8. Jupyter- Very useful, really helps with navigation. Also recommend Nbextensions.
  9. VSCode- Very useful, but I’ve only touched the surface when it comes to VSCode shortcuts, there are many more out there. Since VSCode is a Microsoft creation, many default shortcuts in Explorer work in VSCode too, which is nice.
  10. Git- Just some basic commands I collated when watching a Git tutorial.
  11. Tableau- Just some commands I collated when doing a Tableau Course.
